# @smachado/clean-architecture-baseline

Clean Code - { NodeJS - Typescript }
Repo Base Implemets Clean Architecture

## Requirements
- NodeJS  v20+
- VS Code
- GitHub access

## instalation

```
git clone https://github.com/SaulMachado97/clean-code-nodejs.git
cd clean-code-nodejs
npm install

npm run start-app

```

## setup
 ``` 
 touch .env
 cp .env.example .env
 npm run dev or npm run start:dev
 ```

# deploy
```
npm run test
npm run build
npm run deploy
```
